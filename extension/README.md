# 备考助手采集扩展 · Chrome Extension

在你**已登录**的浏览器里运行,把正在看的学习资料页面采集进备考网站的资料库。不接触任何密码。

## 安装(一次性)
1. 打开 Chrome,地址栏输入 `chrome://extensions`
2. 右上角打开「开发者模式 / Developer mode」
3. 点「加载已解压的扩展程序 / Load unpacked」,选择这个 `extension` 文件夹
4. 扩展出现在工具栏(可固定)

## 使用
1. 到备考网站 → 设置页,复制你的「采集令牌」
2. 点扩展图标,填入网站地址和令牌(只需一次,自动记住)
3. 在资料网站登录,打开要采集的页面
4. 点「采集当前页」抓这一页;或点「自动翻页采集」自动沿"下一页"链接连续抓取
5. 内容进入你当前激活考试的资料库,几秒后在「资料」页可见

## 说明
- 采集写入的是你**当前激活的考试**,先在网站里建好/选好考试。
- 自动翻页靠识别页面里的"下一页/next"链接,不同网站命中率不同;不行就逐页手动采集。
- 只采集你有权访问的页面。请遵守目标网站的服务条款和版权规定。
