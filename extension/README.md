# 备考助手采集扩展 · Chrome Extension (v1.1)

在你**已登录**的浏览器里运行,把学习网站的内容采集进备考网站资料库。不接触任何密码。

## 安装(一次性)
1. Chrome 地址栏输入 `chrome://extensions`
2. 打开右上角「开发者模式 / Developer mode」
3. 点「加载已解压的扩展程序 / Load unpacked」,选这个 `extension` 文件夹

## 通用设置
在备考网站 → 设置页复制「采集令牌」,点扩展图标填入网站地址和令牌(只需一次)。采集会写入你**当前激活的考试**。

## 两种模式
### 手动采集
- 「采集当前页」:抓当前这一页
- 「按下一页翻页采集」:沿页面里的"下一页"链接连续抓取(需指定页数)

### 🤖 Agent 自动(新)
在你已登录的浏览器里,agent 自己观察页面、翻页、采集,像一个真人助手:
1. 切到「Agent 自动」标签
2. 填目标,例如「把这门课所有章节的内容都采集进来」
3. 点「开始」,agent 逐步执行,过程实时显示;随时可「停止」

**安全护栏(硬性)**:agent 只会读取内容和点击导航/翻页/章节类链接。系统提示强制禁止点击任何含"提交/购买/支付/删除/退出/确认订单"等字样的链接,绝不填表、不发消息、不花钱、不改账户状态。拿不准就跳过或结束。即便如此,建议首次在无风险的纯阅读页面上试用,并留意过程日志。

## 说明
- agent 的"下一步"决策由备考网站的 AI 做出(用你的令牌鉴权),点击/采集在你本地浏览器执行,全程不接触你在资料网站的密码。
- 自动化命中率取决于目标网站结构;复杂网站可回退到手动采集。
- 请遵守目标网站的服务条款与版权规定,只采集你有权访问的内容。
